# Hive Checker

![screenshot](./assets/screenshot.png)

Check your own cosmetics on The Hive Minecraft Server.

[Install from Chrome Web Store](https://chromewebstore.google.com/detail/hive-checker/imiaaochonlkmpnmgaoponmbbhfadpdi?hl=ja)

## Manual Installation
1. Clone this repository.
2. Run `pnpm install` to install dependencies.
3. Run `pnpm build` to build the project.
4. Load the ZIP file from the `./build` directory into your browser.
